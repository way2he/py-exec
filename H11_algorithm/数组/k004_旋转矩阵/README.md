# 旋转矩阵

## 题目描述

给你一幅由 **N × N** 矩阵表示的图像，其中每个像素的大小为 4 字节。请你设计一种算法，将图像**旋转 90 度**。

**关键挑战**：能否在**不占用额外内存空间**的情况下完成？

## 提示

- `matrix.length == matrix[i].length == N`
- `1 <= N <= 20`
- `matrix[i][j]` 是一个 32 位有符号整数

## 示例

### 示例 1：

**输入**：
```python
matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]
```

**输出**：
```python
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3]
]
```

### 示例 2：

**输入**：
```python
matrix = [
  [ 5,  1,  9, 11],
  [ 2,  4,  8, 10],
  [13,  3,  6,  7],
  [15, 14, 12, 16]
]
```

**输出**：
```python
[
  [15, 13,  2,  5],
  [14,  3,  4,  1],
  [12,  6,  8,  9],
  [16,  7, 10, 11]
]
```
